## **3Sum – Interview-Ready Explanation**

### **Problem Summary**

Given an integer array `nums`, find **all unique triplets** such that:

```
nums[i] + nums[j] + nums[k] = 0
```

Triplets must be **unique**, and indices must be different.

---

## **Optimal Approach: Sorting + Two Pointers**

### **Why this approach?**

* Brute force is **O(n³)** → too slow
* Sorting allows us to apply the **two-pointer technique**
* Fixing one number reduces the problem to **2Sum**

---

### **Algorithm**

1. **Sort the array**
2. Loop through the array and fix one element `i`
3. For remaining elements, use **two pointers**

   * `left = i + 1`
   * `right = n - 1`
4. If sum is:

   * `0` → valid triplet, store it
   * `< 0` → move `left`
   * `> 0` → move `right`
5. **Skip duplicates** for `i`, `left`, and `right`

---

### **Code (Java)**

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        Arrays.sort(nums);

        for (int i = 0; i < nums.length - 2; i++) {

            // Skip duplicate first element
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int left = i + 1;
            int right = nums.length - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];

                if (sum == 0) {
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));

                    // Skip duplicates
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;

                    left++;
                    right--;
                } 
                else if (sum < 0) {
                    left++;
                } 
                else {
                    right--;
                }
            }
        }
        return result;
    }
}
```

---

## **Time & Space Complexity**

* **Time Complexity:**

  ```
  O(n²)
  ```

  (outer loop + two-pointer scan)

* **Space Complexity:**

  ```
  O(1)
  ```

  (excluding output list)

---

## **Key Interview Points to Mention**

* Sorting enables two pointers
* Duplicate triplets are avoided by skipping repeated elements
* This is an optimized reduction of **3Sum → 2Sum**
